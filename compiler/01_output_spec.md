# Output Specification

Specification for compiled document output format.

## Output Location

All compiled documents are written to:
- `docs/<DOC>/05_draft.md`

Where `<DOC>` is the document folder name.

## Format

Documents should be written in Markdown format.

## Structure

Documents should follow the outline in `01_outline.md`, with:

- Clear headings and subheadings
- Proper Markdown formatting
- Citations for facts (see `universal/06_sources_policy.md`)
- Consistent voice and tone (see `universal/02_voice_tone_style.md`)

## Content Requirements

- Align with `00_doc_goal.md`
- Include all key facts from `02_key_facts.md` (with sources)
- Incorporate dialogue elements from `03_dialogue.md` naturally
- Use snippets from `04_snippets.md` where appropriate
- Follow writing guidelines from `universal/03_writing_guidelines.md`

## Quality Checks

Before finalizing:
- ✅ Goal achieved (check against `00_doc_goal.md`)
- ✅ Outline followed
- ✅ Facts cited or marked TODO
- ✅ Voice/tone consistent
- ✅ Clear call-to-action (if required)
