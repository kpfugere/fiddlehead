# Quick Start Guide

## For Project Creators (Using This Template)

### 1. Run Setup
```bash
./setup.sh
```

### 2. Complete Interview
Answer the questions to generate your universal context document.

### 3. Review Documentation
- Check `docs/universal_context.md` - Your main context document
- Update `SCOPE.md` if needed

## For Agents Working on Projects

### 1. Read Agent Instructions
Start with `AGENTS.md` for workflow and task tracking.

### 2. Reference Universal Context
Read `docs/universal_context.md` for complete project context.

### 3. Check Project Scope
Review `SCOPE.md` for project boundaries.
