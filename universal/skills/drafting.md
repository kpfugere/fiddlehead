# Drafting Skill

Create the initial draft of the document.

## Purpose

Compile all artifacts into a coherent draft following the outline and incorporating facts, dialogue, and snippets.

## Process

1. Review `01_outline.md` for structure
2. Incorporate key facts from `02_key_facts.md`
3. Include dialogue elements from `03_dialogue.md`
4. Use snippets from `04_snippets.md` where appropriate
5. Follow voice/tone from `universal/02_voice_tone_style.md`
6. Write the draft following `compiler/01_output_spec.md`

## Output

Create/update `docs/<DOC>/05_draft.md` with the complete draft.

## Guidelines

- Follow the outline structure
- Maintain consistent voice and tone
- Incorporate facts with proper citations
- Use dialogue elements naturally
- Ensure draft aligns with document goal
