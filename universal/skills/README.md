# Skills Playbooks

Reusable playbooks for common writing tasks.

## Available Skills

- **project_init.md** - Initialization interview to gather universal context
- **outline.md** - Creating and refining document outlines
- **facts.md** - Gathering and verifying facts
- **dialogue.md** - Developing dialogue and conversational elements
- **drafting.md** - Drafting strategies and techniques
- **voice_tone.md** - Maintaining voice and tone consistency
- **editing.md** - Editing and refinement approaches
- **critique.md** - Critique and review processes

## Usage

Refer to these skills from `universal/agents.md` when working through the writing pipeline.
