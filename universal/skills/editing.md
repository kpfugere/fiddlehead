# Editing Skill

Edit and refine the draft for clarity, flow, and impact.

## Purpose

Improve the draft through careful editing and refinement.

## Process

1. Read through the draft for clarity
2. Check flow and transitions
3. Tighten language and remove redundancy
4. Ensure each section supports the document goal
5. Verify facts and citations
6. Check voice/tone consistency

## Output

Update `docs/<DOC>/05_draft.md` with improvements.

## Guidelines

- Keep artifacts small and composable
- Prefer editing existing files over creating new ones
- Focus on clarity and impact
- Maintain consistency with guidelines
