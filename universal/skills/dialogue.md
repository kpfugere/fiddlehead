# Dialogue Skill

Develop dialogue, objections, crisp lines, and conversational elements.

## Purpose

Surface potential objections, craft memorable lines, and develop conversational elements that engage readers.

## Process

1. Identify potential objections readers might have
2. Develop responses to those objections
3. Craft crisp, memorable lines
4. Create examples that illustrate key points
5. Develop analogies or metaphors if helpful

## Output

Update `docs/<DOC>/03_dialogue.md` with:
- Objections and responses
- Key memorable lines
- Examples
- Analogies/metaphors

## Guidelines

- Address objections preemptively when possible
- Keep lines concise and memorable
- Use examples that resonate with the audience
- Ensure dialogue elements support the document goal
